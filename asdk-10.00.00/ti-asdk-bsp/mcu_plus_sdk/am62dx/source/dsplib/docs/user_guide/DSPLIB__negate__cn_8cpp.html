This function is the main execution function for the natural C implementation of the kernel. The function declaration conforms to the declaration of DSPLIB_negate_exec.

Parameters
[in]handle: Active handle to the kernel
[in]pIn: Pointer to buffer holding the input data
[out]pOut: Pointer to buffer holding the output data
