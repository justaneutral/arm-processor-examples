VXLIB_accumulateWeightImage_priv.h File Reference

Introduction

Header file for kernel's internal use. For the kernel's interface. Note: The VXLIB_accumulateWeightImage does not contain any implementation. Instead, the implementation of VXLIB_addWeight is used. For this reason, the kerenl's interface of VXLIB_addWeight (defined in VXLIB_addWeight_priv.h) are reused in this kernel. please see VXLIB_accumulateWeightImage.
