◆ VXLIB_histogram_exec_scalar_ci()

template<typename dTypeIn , typename dTypeOut >
VXLIB_STATUS VXLIB_histogram_exec_scalar_ci ( VXLIB_kernelHandle  handle,
void *restrict  pIn,
void *restrict  pOut,
void *restrict  pTemp 
)

This function is the main execution function for the C7504 implementation of the kernel. The function declaration conforms to the declaration of VXLIB_histogram_exec. The following kernel uses C66 code (scalar) due to C7504 lack of hardware LUT histogram.

Parameters
[in]handle: Active handle to the kernel
[in]pIn: Pointer to buffer holding the input image
[out]pOut: Pointer to buffer holding the output data
[out]pTemp: Pointer to buffer holding the scratch buffer data
Returns
Status value indicating success or failure. Refer to VXLIB_STATUS.
Performance Considerations:
For best performance,
  • the input and output data images are expected to be in L2 memory
  • the buffer pointers are assumed to be 64-byte aligned
