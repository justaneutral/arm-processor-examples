This function is the main execution function for the natural C implementation of the kernel. The function declaration conforms to the declaration of VXLIB_meanStdDev_exec.

Parameters
[in]handle: Active handle to the kernel
[in]pIn0: Pointer to buffer holding the input image
[in,out]pPixelsProcessed: Pointer to number of pixels processed
[in,out]pCurrentSum: Pointer to current sum accumulation
[in,out]pCurrentSqSum: Pointer to current square sum accumulation
[in,out]pOut0: Pointer to output holding mean
[in,out]pOut1: Pointer to output holding standard deviation
Returns
Status value indicating success or failure. Refer to VXLIB_STATUS.
