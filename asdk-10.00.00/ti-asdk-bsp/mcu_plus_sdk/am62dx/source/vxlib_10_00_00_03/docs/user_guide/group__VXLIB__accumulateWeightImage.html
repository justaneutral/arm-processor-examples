VXLIB_accumulateWeightImage
VXLIB kernels

Introduction

Kernel for weighted accumulation of input image and output image. This kernel has no internel implementation. Instead,it calls the VXLIB_addWeight kernel implementation.

Performs weighted accumulation of the input image to the output image. The following datatype combinations are supported:

Case Input Output
0 uint8_t uint8_t
Method
dst(x,y) = alpha * src(x,y) + (1 - alpha) * dst(x,y); where 0.0 <= alpha <= 1.0

Please refer to details under VXLIB_accumulateWeightImage_exec

Parameters
[in]handle: Active handle to the kernel
[in]pIn: Pointer to the structure input image
[out]pOut: Pointer to the output image
[in]alpha: Alpha scalar input
Returns
Status value indicating success or failure. Refer to VXLIB_STATUS.
Assumptions:
  • None
Performance Considerations:
For best performance,
  • the input and output data buffers are expected to be in L2 memory
  • the buffer pointers are assumed to be 64-byte aligned
Remarks
Before calling this function, application is expected to call VXLIB_accumulateWeightImage_init and VXLIB_accumulateWeightImage_exec_checkParams functions. This ensures resource configuration and error checks are done only once for several invocations of this function.
