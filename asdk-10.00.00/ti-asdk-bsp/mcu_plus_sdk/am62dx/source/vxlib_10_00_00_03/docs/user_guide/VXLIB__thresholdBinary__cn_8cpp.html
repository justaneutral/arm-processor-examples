VXLIB_STATUS VXLIB_thresholdBinary_exec_cn ( VXLIB_kernelHandle  handle,
void *restrict  pIn,
void *restrict  pOut,
void *restrict  pThresholdVal,
void *restrict  pTrueVal,
void *restrict  pFalseVal 
)

This function is the main execution function for the natural C implementation of the kernel. The function declaration conforms to the declaration of VXLIB_thresholdBinary_exec.

Parameters
[in]handle: Active handle to the kernel
[in]pIn: Pointer to buffer holding the input image
[out]pOut: Pointer to buffer holding the output data
[in]pThresholdVal: Pointer to the threshold value
[in]pTrueVal: Pointer to the true value
[in]pFalseVal: Pointer to the false value
Returns
Status value indicating success or failure. Refer to VXLIB_STATUS.
