◆ VXLIB_minMaxLoc_init_ci()

template<uint32_t dType>
VXLIB_STATUS VXLIB_minMaxLoc_init_ci ( VXLIB_kernelHandle  handle,
VXLIB_bufParams2D_t *  bufParamsIn,
VXLIB_bufParams1D_t *  bufParamsMinLoc,
VXLIB_bufParams1D_t *  bufParamsMaxLoc,
const VXLIB_minMaxLoc_InitArgs *  pKerInitArgs 
)

This function is the initialization function for the C7x implementation of the kernel. The function declaration conforms to the declaration of VXLIB_minMaxLoc_init.

This function determines the configuration for the streaming engine resources based on the function call parameters, and the configuration is saved in bufPBlock array. In the kernel call sequence, VXLIB_minMaxLoc_exec_ci would be called later independently by the application. When VXLIB_minMaxLoc_exec_ci runs, it merely retrieves the configuration from the bufPBlock and uses it to set up the hardware resources. This arrangement is so that VXLIB_minMaxLoc_exec_ci does not lose cycles to determine the hardware configuration.

Parameters
[in]handle: Active handle to the kernel
[in]bufParamsIn: Pointer to the structure containing dimensional information of input image
[out]bufParamsMinLoc: Pointer to the structure containing dimensional information of minLoc
[out]bufParamsMaxLoc: Pointer to the structure containing dimensional information of maxLoc
[in]pKerInitArgs: Pointer to the structure holding init parameters
Returns
Status value indicating success or failure. Refer to VXLIB_STATUS.

This function is the main execution function for the C7x implementation of the kernel. The function declaration conforms to the declaration of VXLIB_minMaxLoc_exec.

Parameters
[in]handle: Active handle to the kernel
[in]pIn[]: Pointer to array containing input image
[in,out]pMinVal: Pointer to Minimum value in the image
[in,out]pMaxVal: Pointer to Maximum value in the image
[in,out]pMinCount: Ponter to total number of detected minimum values in the image (optional)
[in,out]pMaxCount: Ponter to total number of detected maximum values in the image (optional)
[in]pMinLocCapacity: Allocated array size of minLoc array.
[in]pMaxLocCapacity: Allocated array size of maxLoc array.
[in,out]pMinLoc[]: Pointer to coordinates of minimum values in the image (x,y) (optional)
[in,out]pMaxLoc[]: Pointer to coordinates of maximum values in the image (x,y) (optional)
[in]pStartX: Starting x coordinate in the image corresponding to the src pointer
[in]pStartY: Starting y coordinate in the image corresponding to the src pointer
Returns
Status value indicating success or failure. Refer to VXLIB_STATUS.
Performance Considerations:
For best performance,
  • the input and output data images are expected to be in L2 memory
  • the buffer pointers are assumed to be 64-byte aligned
