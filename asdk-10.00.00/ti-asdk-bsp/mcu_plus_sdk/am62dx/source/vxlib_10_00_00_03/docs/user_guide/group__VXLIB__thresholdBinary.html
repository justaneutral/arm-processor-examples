VXLIB_thresholdBinary
VXLIB kernels

Introduction

Kernel for Thresholding an input image and produces an output Boolean image.

The following datatype combinations are supported:

Case Input Output
1 uint8_t uint8_t
2 int8_t int8_t
3 uint16_t uint16_t
2 int16_t int16_t
Method
  dst(x,y) = trueValue    if src(x,y) > threshold
  dst(x,y) = falseValue   otherwise

OpenVX Function Reference Page : https://www.khronos.org/registry/vx/specs/1.1/html/d3/d1e/group__group__vision__function__threshold.html

This function is the main kernel compute function.

Please refer to details under VXLIB_thresholdBinary_exec

Parameters
[in]handle: Active handle to the kernel
[in]pIn: Pointer to the structure input image
[out]pOut: Pointer to the output image
[in]pThresholdVal: Pointer to the threshold value
[in]pTrueVal: Pointer to the true value
[in]pFalseVal: Pointer to the false value
Returns
Status value indicating success or failure. Refer to VXLIB_STATUS.
Assumptions:
  • None
Performance Considerations:
For best performance,
  • The input and output data buffers are expected to be in L2 memory
  • The buffer pointers are assumed to be 64-byte aligned
Remarks
Before calling this function, application is expected to call VXLIB_thresholdBinary_init and VXLIB_thresholdBinary_exec_checkParams functions. This ensures resource configuration and error checks are done only once for several invocations of this function.
