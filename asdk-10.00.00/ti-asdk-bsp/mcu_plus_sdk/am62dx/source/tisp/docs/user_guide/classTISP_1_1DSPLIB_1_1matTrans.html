template<typename T, dmaDir_t dmaDir>
class TISP::DSPLIB::matTrans< T, dmaDir >

Provides node-level APIs for the DSPLIB_matTrans kernel from DSPLIB.

Template Parameters
T: Supports float and double
dmaDir: Supports TISP::NO_DMA, TISP::DMA_IN, TISP::DMA_OUT, and TISP::DMA_IN_OUT
