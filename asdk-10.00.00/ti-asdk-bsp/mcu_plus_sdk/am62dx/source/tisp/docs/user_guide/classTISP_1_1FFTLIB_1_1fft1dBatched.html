Detailed Description

template<typename T, dmaDir_t dmaDir>
class TISP::FFTLIB::fft1dBatched< T, dmaDir >

Provides node-level APIs for the FFTLIB_fft1dBatched kernel from DSPLIB. Assumes complex float data that is stored in interleaved fashion.

Template Parameters
T: Supports float
dmaDir: Supports TISP::NO_DMA, TISP::DMA_IN, TISP::DMA_OUT, and TISP::DMA_IN_OUT
