template<typename T, dmaDir_t dmaDir>
class TISP::DSPLIB::addConstant< T, dmaDir >

Provides node-level APIs for the DSPLIB_addConstant kernel from DSPLIB.

Template Parameters
T: Supports float
dmaDir: Supports TISP::NO_DMA, TISP::DMA_IN, TISP::DMA_OUT, and TISP::DMA_IN_OUT
