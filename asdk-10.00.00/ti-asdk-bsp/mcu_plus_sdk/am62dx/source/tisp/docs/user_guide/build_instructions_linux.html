TISP User Guide
Ubuntu 22.04 Build Instructions for TISP
Environment Setup
  • Run the executables on EVM using CCS (Code composer studio)
  • Setup needed for the compiler for CGT7x
    • Download CGT7x Compiler Installer and install
    • export CGT7X_ROOT=${CGT7X_COMPILER_DIR}
  • Setup MCU_PLUS_SDK_ROOT environment variable
    • export MCU_PLUS_SDK_ROOT=${MCU_PLUS_SDK_ROOT_DIR}
    • Example : export MCU_PLUS_SDK_ROOT=${HOME}/mcu_plus_sdk_am62ax_10_00_00_10
  • Path setup
    • export PATH=${CGT7X_ROOT}/bin:${PATH}
Build
  • TISP utilizes CMake to build, ensure that system has CMake v3.16.0 or higher
  • Build options:
    • SOC=AM62A/j722s/AM62D
    • DEVICE=C7504/C7524/C7504
    • TARGET_PLATFORM=PC/""
    • KERNEL_NAME
    • BUILD_TEST=1 OR BUILD_EXAMPLE=1 OR AUTO_TEST=1
    • TISP_DEBUGPRINT=1/0
    • CMAKE_EXPORT_COMPILE_COMMANDS=TRUE
    • CMAKE_BUILD_TYPE=Release/Debug
    • AUTO_TEST=1/0 (builds all test cases and libraries)
  • Example CMake commands
    • To build test/FFTLIB/TISP_fft2d_superNode for target on AM62A and run performance test cases:
      • cmake -B build -DTARGET_PLATFORM="" -DBUILD_TEST="1" -DKERNEL_NAME="FFTLIB/TISP_fft2d_superNode" -DSOC="AM62A" \
        -DDEVICE="C7504" -DTISP_DEBUGPRINT="0" -DALL_TEST_CASES="1" -DCMAKE_EXPORT_COMPILE_COMMANDS="TRUE" \
        -DCMAKE_BUILD_TYPE="Release"
    • To build test/misc/TISP_blockCopy_superNode for host emulation on AM62A and with debug print statements
      • cmake -B build -DTARGET_PLATFORM="PC" -DBUILD_TEST="1" -DKERNEL_NAME="misc/TISP_blockCopy_superNode" -DSOC="AM62A" \
        -DDEVICE="C7504" -DTISP_DEBUGPRINT="1" -DCMAKE_EXPORT_COMPILE_COMMANDS="TRUE" \
        -DCMAKE_BUILD_TYPE="Release"
    • To build all tests cases and libraries for release for AM62A for target
      • cmake -B build -DTARGET_PLATFORM="" -DAUTO_TEST="1" -DSOC="AM62A" -DDEVICE="C7504" \
        -DALL_TEST_CASES="1" -DCMAKE_BUILD_TYPE="Release" -DTISP_DEBUGPRINT="0"
    • Then run:
      • cmake --build build -j<num_cores>
Run
  • To run TISP kernels on a PC:
    • bin/Release/test_TISP_<kernel_name>_<device>_x86_64
      OR
      bin/Debug/test_TISP_<kernel_name>_<device>_x86_64
      • Ex:
        bin/Release/test_TISP_and_C7504_x86_64
  • To run TISP kernels for target
    • To reproduce numbers given in the datasheet, run the .out program on the EVM in OSPI NAND Boot Mode
